# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## Requirements
 -Node.js 18, 
 -Vite 5, 
 -React 18, 
 -React-Bootstrap 2

## Installation
```bash
npm install
npm install react-bootstrap bootstrap

To start the project, you can run:
npm run dev
